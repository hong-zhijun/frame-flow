# FrameFlow 产品路线图

## V1 — 核心查看器

### 1. 文件夹导入与浏览
- 选择文件夹导入，按目录结构浏览图片
- 支持嵌套子文件夹

### 2. 历史文件夹记录
- 记住打开过的文件夹，下次启动可快速访问
- 支持清除历史

### 3. 滚轮缩放
- 以鼠标指针位置为中心缩放
- 支持拖拽平移
- 双击还原

### 4. 快捷翻页
- 空格键 / 方向键切换上下张
- 支持左右滑动手势

### 5. RAW 格式支持
- 支持 CR2 / NEF / ARW / DNG / RAF 等主流 RAW 格式
- 基于 macOS 系统 ImageIO 原生解码

---

## V2 — 标记与导出

### 1. 星级标记与筛选
- 1-5 星评级
- 按星级筛选显示
- 标记数据本地持久化（全局 JSON 文件，key 为文件路径）
- 打开文件夹时自动清理无效条目（文件已不存在的评级记录）

### 2. 星级归档
- 按星级筛选后，一键归档到新文件夹（如「归档-五星」）
- 在当前文件夹下创建归档文件夹，移动对应文件
- 自动处理文件名冲突（同名文件加后缀）
- 移动后自动刷新图片列表

### 3. 图片套壳导出
- 读取 EXIF 元数据（设备型号、镜头参数、GPS 地点等）
- 底栏展示设备 Logo、型号、拍摄参数、地点、作者
- 支持自定义编辑套壳内容
- 一键导出带壳图片

---

## 技术栈

- **语言**: Swift
- **UI 框架**: SwiftUI
- **图片解码**: ImageIO / Core Image（原生 RAW 支持）
- **数据存储**: UserDefaults / 本地 JSON
- **平台**: macOS 原生应用
- **界面语言**: 中文（所有按钮、标签、提示文字均使用中文）
